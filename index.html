About Me

My name is Amit Ranganathan and I am currently an incoming junior at San Jose State University in the Bay Area. I am a computer science major and absolutely love what I do. I was first introduced to the world of computer science when I took a C++ class at De Anza community college. It was interesting, but it didn't really grab my attention. However, after I took AP Computer Science in my senior year of high school, I fell in love with the logic behind programming. I was never really good at biology or chemistry or any other science for that matter, but I was great at computer science. Then when I entered college, things just got better and better as I was learning a lot of knew information and I was excelling in all my classes. I knew I had made the right decision in pursuing computer science and cannot wait to see what the future has in store for me.

Projects

Whiteboard GUI - JAVA

Created a whiteboard user interface using the Java swing framework, along with Java AWT. Features include being able to draw certain shapes, resizing them, changing their colors, and other various features. Another feature is save/open, where I use serialization to write out an XML file of the current whiteboard instance, which can then be opened to display that instance. Finally, I used networking to create a client and server connection.

Arithmetic Calculator - MIPS 32 bit assembly

Designed and developed an arithmetic calculator, which included addition, subtraction, multiplication, and division operations, using assembly level bitwise logical operations.
