About Me

Who Am I?

My name is Amit Ranganathan and I am currently an incoming junior at San Jose State University in the Bay Area. I am a computer science major and absolutely love what I do. I realized I wanted to pursue computer science in the summer before my junior year of high school when I took a C++ class. I fell in love with the logic behind programming and knew then itself that CS is what I wanted to pursue.

Projects

Whiteboard GUI - JAVA

Created a whiteboard user interface using the Java swing interface, along with Java AWT. Features include being able to draw certain shapes, resizing them, changing their colors, and other various features. Another feature is save/open, where I use serialization to write out an XML file of the current whiteboard instance, which can then be opened to display that instance. Finally, I used networking to create a client and server connection.

Arithmetic Calculator - MIPS 32 bit assembly

Designed and developed an arithmetic calculator, which included addition, subtraction, multiplication, and division operations, using assembly level bitwise logical operations.
